# ☁️ CloudVault

## Project Overview
CloudVault is a secure cloud-based file storage web application developed using Flask and Firebase Authentication. It allows users to register, log in, upload, download, search, and delete files through a simple and user-friendly interface.

## Features
- User Registration & Login
- Firebase Email Authentication
- File Upload
- File Download
- File Delete
- Search Files
- Dark Mode
- Responsive Design
- User Logout

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Firebase Authentication
- Git & GitHub

## Installation
1. Clone the repository.
2. Install requirements:
   ```
   pip install -r requirements.txt
   ```
3. Run:
   ```
   python app.py
   ```
4. Open:
   ```
   http://127.0.0.1:5000
   ```
